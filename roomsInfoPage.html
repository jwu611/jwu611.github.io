About

Development

Rooms is my first experience using Unity and programming in C#. The premise for Rooms was based on classic 2D online room escape games such as Neutral Escape Games and Afro-Ninja's Escape Series. The first (and currently only) level includes puzzles revolving around the theme of "monochrome". Thus, there are puzzles involving sheet music, chess, black and white clocks, and light and dark. Main development took place over about 20 days, from late July to mid-Augest 2020

All puzzles and artwork are my own design and work. Artwork was done using stock images and edited in GIMP.

Known Issues

 - A new king piece reappears in number code box every time the box is opened, even after king piece has been collected
 - Knight piece cannot be used on chess board if more than 1 king piece (from issue #1) is in inventory
 - Knight piece disappears from ground if it is not collected before number code box is unlocked
